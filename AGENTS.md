# 红韵智学 - 党政学习全屏沉浸平台

## 产品定位

**全屏沉浸式党政学习平台**，像刷抖音一样学习。
- 整合权威学习资源
- 智能推荐精品内容
- 让学习更高效、更有趣

## 技术栈

- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Language**: TypeScript 5
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **Styling**: Tailwind CSS 4
- **字体**: Noto Serif SC

## 页面结构

```
├── /                    # 首页 - 全屏刷内容流
├── /library             # 知识库 - 系统课程浏览
└── /api/llm             # LLM API 路由（内容转化用）
```

## 核心功能

### 1. 首页 - 全屏刷内容流
- **全屏沉浸模式**：单列全屏、无限下滑
- **内容类型**：
  - 微课（1-3分钟短视频）
  - 金句（图文海报，停留3秒）
  - 长课（完整讲座，可加入书架）
- **交互**：滚轮/上滑切换下一个
- **自动播放**：进入即自动播放
- **频道Tab**：推荐、时政、党史、实务、我的课表

### 2. 课程关联机制
- 内容结束时弹出关联课程卡片
- 右侧滑出课程目录抽屉
- 已学章节打钩，未学章节显示时长
- 保留课程完整性

### 3. 知识胶囊
- 视频/文章中自动识别关键知识点
- 屏幕侧边浮现知识点卡片
- 无需暂停，扫一眼即可

### 4. 进度管理
- 断点续传
- 智能续播
- 顶部悬浮"继续学习"条

### 5. 知识库 (`/library`)
- 系统课程浏览
- 微课速学
- 金句收藏
- 分类筛选

## 开发命令

```bash
# 安装依赖
pnpm install

# 开发环境
pnpm dev

# 构建生产版本
pnpm build

# 代码检查
pnpm lint
```

## 注意事项

1. 首页采用全屏沉浸式设计，无传统导航
2. 底部固定导航栏
3. 使用 Noto Serif SC 字体
4. 主题采用党政风格的红橙渐变色系
