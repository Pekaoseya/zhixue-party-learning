# 红韵智创 - 党政内容创作者专属平台

## 项目概述

红韵智创是一个集权威学习、智能创作、新媒体赋能于一体的党政内容创作者专属平台。

## 技术栈

- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Language**: TypeScript 5
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **Styling**: Tailwind CSS 4
- **AI能力**: coze-coding-dev-sdk (LLM)

## 页面结构

```
├── /                    # 首页
├── /library             # 权威智库模块
├── /create              # 智能创作模块 (含 AI 公文助手)
├── /media               # 新媒体赋能模块
└── /api/llm             # LLM API 路由
```

## 核心功能模块

### 1. 权威智库模块 (/library)
- 权威资讯聚合：整合学习强国、人民日报、求是网等资源
- 精品范文库：提供各类公文范文参考
- 理论溯源工具：查询政治术语、名言警句的出处

### 2. 智能创作模块 (/create)
- **AI公文助手**：基于LLM生成各类公文初稿
- **智能校对润色**：自动检查错别字、语法错误
- **灵感激发器**：推荐金句、案例和数据

### 3. 新媒体赋能模块 (/media)
- **政务排版工具**：AI一键排版
- **数据可视化**：生成柱状图、饼图等
- **多平台分发**：一键发布至微信公众号等

## 开发命令

```bash
# 安装依赖
pnpm install

# 开发环境
pnpm dev

# 构建生产版本
pnpm build

# 代码检查
pnpm lint
```

## 注意事项

1. AI公文助手通过 `/api/llm` 路由调用LLM服务
2. 使用 `noto-serif-sc` 字体作为宋体备选
3. 主题采用党政风格的红橙渐变色系
